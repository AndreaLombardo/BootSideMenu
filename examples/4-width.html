The menu is on the left with a 360px width.

$(document).ready(function () {
    $('#test').BootSideMenu({
        side: "left",
        pushBody:false,
        width: '360px'
    });
});

By default when you click a menu's item it will close automatically. To change this feature use the option:

closeOnClick:false
